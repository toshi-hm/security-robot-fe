# 操作説明書 - はじめに

このドキュメントは、セキュリティロボット強化学習システムのフロントエンドアプリケーションの操作方法について説明します。

本システムは、強化学習を用いて警備ロボットの最適な移動経路をシミュレーションするためのWebアプリケーションです。
ユーザーは、様々なパラメータを設定して学習セッションを実行し、その結果をリアルタイムで可視化・分析したり、過去の学習結果を再生して確認したりすることができます。

## 各機能の操作説明

以下のリンクから、各機能の詳細な操作説明を参照してください。

*   **[1. ダッシュボード](./01_dashboard.md)**
    *   システムの全体像を把握するためのメインページです。

*   **[2. 学習 (Training)](./02_training.md)**
    *   新しい強化学習セッションの開始、実行中のセッションの監視、詳細なメトリクスの確認ができます。

*   **[3. 再生 (Playback)](./03_playback.md)**
    *   完了した学習セッションの様子をステップごとに再生し、ロボットの行動や環境の変化を分析できます。

*   **[4. モデル管理 (Models)](./04_models.md)**
    *   学習によって生成されたモデルのアップロード、ダウンロード、削除ができます。

*   **[5. 設定 (Settings)](./05_settings.md)**
    *   学習アルゴリズムやシミュレーション環境に関する様々なパラメータを設定・保存できます。

*   **[6. 開発者向け情報](./06_developer_info.md)**
    *   APIテストページなど、開発者向けの機能について説明します。
